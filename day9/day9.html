What is an iFrame? An iFrame is an HTML element that enables an inline frame for the embedding of external content. Essentially, you can load another web page within a designated area of your current webpage.

preload attribute ko simple hinglish mein samjhaata hoon — ye audio (ya video) tag ke andar hota hai aur browser ko yeh batata hai ki audio file ko page load ke time kitna load karna chahiye.

none 🟡 Matlab: Kuch bhi preload mat karo. ➡️ Browser audio file bilkul bhi download nahi karega jab tak user play button nahi dabata. ✅ Kab use karein: Jab tu bandwidth bachana chahta hai ya user se pehle action chahta hai.

auto 🔵 Matlab: Browser apne aap decide karega ki file kitni preload karni hai — aur generally zyada preload karta hai. ➡️ Ye file background mein load karta hai taaki user play kare to instantly start ho jaye. ✅ Kab use karein: Jab tu chahata hai ki audio jaldi play ho jaaye bina rukawat ke.

metadata 🟢 Matlab: Sirf audio file ki basic info jaise duration, size wagairah load karo. ➡️ Puri audio file load nahi hoti. ✅ Kab use karein: Jab tujhe sirf info dikhani hai jaise "song is 3:45 long", bina pura song load kiye.
